# 前 n 个数字二进制中 1 的个数

给定一个非负整数 n ，请计算 0 到 n 之间的每个数字的二进制表示中 1 的个数，并输出一个数组。

```javascript
var countBits = function(n) {
    const dp = new Array(n + 1);
    dp[0] = 0;// 初始化数组
    for (let i = 1; i <= n; i++) {
        // 如果是偶数，1的个数一定和除以2后的数的1的个数相同；
        // 如果是奇数, 等于上一个数+1（dp[i-1]+1）
        dp[i] = i % 2 ? dp[i - 1] + 1 : dp[i >> 1];
    }
    return dp;
};

```
